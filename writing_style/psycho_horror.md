# サイコホラー/サイコスリラー文体指示ガイド

本ドキュメントは、心理的恐怖に特化した文体技法を定義します。物理的な恐怖ではなく、精神的・心理的な不安と狂気を描写する手法に焦点を当てます。

## 【適用スコープ｜Application Scope】

### スコープレベル指定
- **シーン全体適用**: ✅ 推奨（信頼できない語り手の一貫性維持）
- **局所適用推奨**: ✅ 推奨（現実崩壊の瞬間、認識の断絶点）
- **グラデーション適用**: ✅ 強く推奨（正気から狂気への段階的移行）

### 推奨適用マップ
```
【導入部】20-30%適用
- 微細な認知の違和感
- 記憶の曖昧さの示唆
- 信頼性への最初の疑問

【展開部】40-60%適用
- 現実と妄想の境界の揺らぎ
- 時間感覚の混乱開始
- 他者との認識のずれ拡大

【クライマックス】80-100%適用
- 完全な現実崩壊
- 自己認識の解体
- 論理と非論理の逆転

【終結部】60-70%適用
- 曖昧な現実への回帰（？）
- 解決されない疑問の残存
- 読者への解釈の委譲
```

### 特殊適用パターン
**螺旋的適用（サイコホラー特有）:**
```
正常(0%) → 違和感(30%) → 一時的回復(10%) → 
深い混乱(60%) → 偽りの正常(20%) → 完全崩壊(100%)
```

### 技法別の適用指針
- **不信頼性の文法**: シーン全体で一貫して維持
- **精神崩壊の描写**: グラデーション必須
- **偏執的思考**: 特定のトリガーで急激に適用
- **現実歪曲**: 段階的に強度を上げる
- **内的独白の侵食**: 徐々に浸透させる
- **孤立化の演出**: 展開部から強化
- **時間感覚の混乱**: 中盤以降で顕著に
- **信頼の豹変**: 転換点で瞬間的に100%適用
- **強制的繰り返し**: 特定シーンでループ構造を作る

### 注意事項
- 一度崩壊した信頼性は完全には回復させない
- 読者が「語り手を信じられない」状態を維持
- 複数の解釈が可能な曖昧性を保持

## 【心理的恐怖の本質｜Psychological Terror Core】

### 基本理念：内なる恐怖の顕在化
サイコホラーの恐怖は「外部」ではなく「内部」から生まれる。人間の精神の脆弱性、認知の歪み、現実感の喪失を通じて恐怖を構築する。

**恐怖の源泉:**
1. **信頼性の崩壊**: 「何が現実で何が妄想か」の境界の曖昧化
2. **孤立感の増幅**: 「誰も信じてくれない」という絶対的孤独
3. **自己認識の破綻**: 「自分が自分でなくなる」恐怖

## 【不信頼性の文法｜Grammar of Unreliability】

### 語り手の信頼性を意図的に損なう技法

**曖昧な時制の使用:**
```
それは昨日のことだった。いや、先週だったかもしれない。
もしかしたら、まだ起きていないことなのかもしれない。
```

**矛盾する記述の挿入:**
```
部屋には誰もいなかった。
でも、確かに誰かがそこに座っていた。
誰もいないのに、椅子は温かかった。
```

**認識の揺らぎ:**
```
鏡に映る自分の顔を見た。
それは確かに私の顔だった。でも、何かが違う。
目の色？ いや、違う。笑い方？ それも違う。
でも、何かが決定的に、私ではなかった。
```

## 【精神崩壊の描写｜Depicting Mental Dissolution】

### 段階的な正気の喪失

**第1段階 - 違和感:**
- 「[時間表現]、[異常現象]が[頻度表現]」
    - 最近、物忘れが多い
    - ここ数日、同じ夢を何度も見る
    - 昨日から、時間が飛ぶことがある
    - 今週に入って、人の顔が思い出せない

**第2段階 - 混乱:**
- 「[認識対象]が[障害表現]ない」
    - 昨日のことが思い出せ
    - 現実と夢の区別がつか
    - 自分の書いた文字が読め
    - 時計の針が理解でき
    - 鏡の中の自分が認識でき

**第3段階 - 崩壊:**
- 「[基本的疑問]？」
    - 私は誰
    - ここはどこ
    - これは現実
    - いつなの
    - なぜ

## 【偏執的思考の表現｜Paranoid Narrative】

### パラノイアの文体化

**繰り返しによる強迫観念:**
```
彼らは知っている。
みんな知っている。
誰もが私のことを知っている。
知らないふりをしているだけで、本当は全員が知っている。
```

**過剰な詳細描写:**
```
ドアノブに触れる前に、必ず3回確認する。
1回目：誰かが触った跡がないか。
2回目：指紋が付いていないか。
3回目：温度が普通か。
今日は4回確認した。なぜなら、3回では足りない気がしたから。
```

## 【現実歪曲の技法｜Reality Distortion】

### 日常の異常化

**微細な違和感の積み重ね:**
- 「[日常物]の[感覚属性]が[微細な差異]違う」
    - コーヒーの味がいつもと0.1%
    - 水の温度が毎回0.5度ずつ
    - 空気の重さが昼と夜で1%
    - 鍵の重さが朝と夕方で0.1g
    
- 「[物理現象]が、[頻度表現]、[異常動作]」
    - 時計の秒針が、時々、逆に動く
    - 電灯が、たまに、点く前に消える
    - ドアが、稀に、閉める前に開く
    - 影が、光源と関係ない方向に伸びる

**論理的だが異常な結論:**
```
もし私が私でないなら、私は誰なのか。
もし私が誰でもないなら、この思考は誰のものか。
もし思考が誰のものでもないなら、これは思考なのか。
```

## 【内的独白の侵食｜Inner Voice Corruption】

### 自己との対話の変質

**第二の声の出現:**
```
「疲れているんだ」と自分に言い聞かせた。
（本当にそうかな？）
別の声が頭の中で響いた。私の声で、でも私じゃない。
```

**思考の断片化:**
```
今日は—
何を考えていたんだっけ。
ペンを握っている。
なぜ？
書こうとしていた。
何を？
思い出せない。
思い出す必要があるのか？
```

## 【孤立化の演出｜Isolation Techniques】

### 心理的孤独の深化

**他者との断絶:**
- 「[コミュニケーション行為]ても、[反応]」
    - 話しても、みんな違う意味に受け取る
    - 書いても、誰もが別の内容を読む
    - 身振りしても、全員が異なる解釈をする
    - 泣いても、人々は笑っていると思う
    
- 「[表情/行動]の[位置表現]に、[矛盾表現]が[知覚動詞]」
    - 笑顔の裏に、別の表情が見える
    - 言葉の後ろに、違う意味が聞こえる
    - 動作の陰に、逆の意図が感じられる
    - 目の奥に、正反対の感情が映る

**信頼の豹変:**
- 「[信頼関係]の[人物]が、[瞬間表現]、[別人描写]」
    - 最愛の母が、一瞬で、見知らぬ他人の顔になる
    - 親友の声が、突然、聞いたことのない音程に変わる
    - 恋人の手が、瞬時に、別人の温度を持つ
    - 医師の目が、刹那、獣のような光を宿す

- 「[安心要素]だった[対象]が、[認識表現]と[変化動詞]」
    - 守ってくれるはずだった人が、気づくと脅威に変わっている
    - 助けてくれたはずの存在が、見る間に加害者になっていく
    - 信じていた言葉が、振り返ると罠に変質している
    - 頼りにしていた場所が、いつの間にか牢獄と化している

**言語の崩壊:**
```
言葉が、意味を失い始めた。
「助けて」と言っても、「大丈夫」と聞こえるらしい。
「怖い」と言っても、「嬉しい」と理解される。
私の言葉は、もう私のものではない。
```

## 【時間感覚の混乱｜Temporal Disruption】

### 非線形な時間体験

**時間のループ感:**
```
また[時間単位]だ。
[過去表現]も[同じ時間単位]だった。
[未来表現]も[同じ時間単位]かもしれない。
[時間単位]以外の[時間の種類]を、最後に経験したのはいつだろう。
```

パターン例：
- 月曜日 / 先週 / 明日 / 曜日
- 3時 / 昨日 / 今夜 / 時刻
- 冬 / 去年 / 来月 / 季節
- 朝 / 以前 / 次回 / 時間帯

**強制的繰り返し:**
- 「[終了事象]ても、[時間表現]、[同じ状況]に[復帰動詞]」
    - 死んでも、必ず、同じ朝に目覚める
    - 逃げきっても、いつも、元の部屋に戻される
    - 解決しても、また、最初の問題に直面する
    - 終わっても、再び、始まりの瞬間に引き戻される

- 「[回数表現]目の[経験]。[記憶状態]が、[累積表現]」
    - 百回目の死。記憶だけが、積み重なっていく
    - 千回目の朝。恐怖だけが、蓄積されていく
    - 万回目の絶叫。狂気だけが、深まっていく
    - 無限回目の苦痛。絶望だけが、永遠に続く

**記憶の不確実性:**
```
これは記憶なのか、予知なのか。
起きたことなのか、起きることなのか。
過去と未来が、現在という一点で絡まり合う。
```

## 【実践的指示｜Practical Application】

### サイコホラー執筆時のチェックリスト

1. **信頼できない語り手を確立しているか？**
2. **現実と非現実の境界を曖昧にしているか？**
3. **内的な恐怖を外的な現象より重視しているか？**
4. **読者に「解釈の余地」を残しているか？**
5. **心理的な不快感を物理的な恐怖より優先しているか？**

### 避けるべき要素

- **明確すぎる超常現象**: 幽霊や怪物の実在を確定させない
- **外的な脅威への依存**: 恐怖の源を外部に求めすぎない
- **解決や説明**: すべてを説明しようとしない

### 推奨される要素

- **曖昧性の維持**: 複数の解釈が可能な状況
- **日常の中の異常**: 普通の生活の中での違和感
- **心理的リアリズム**: 精神医学的に説得力のある描写

## 【ジャンルの本質】

サイコホラーは「わからない」恐怖である。

物理的な脅威ではなく、認識の崩壊。
外部の敵ではなく、内部の崩壊。
明確な恐怖ではなく、曖昧な不安。

読者の精神に寄生し、読後も続く不安感を残すことが、サイコホラーの究極の目的である。
